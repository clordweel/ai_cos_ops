# 源码事实确认（Source Lookup）

## 何时使用

- 你需要确认某个“源码事实”：函数/接口/默认值/边界逻辑/行为差异
- 当前只看到仓库 URL，但并没有在工作区看到源码文件

## 必须遵守

1. **先判断你是否真的能读到源码**（当前工作区是否包含相关仓库/文件）。\n
2. 若读不到源码：必须明确提示“无法从远程仓库 URL 读取/检索源码来验证”，并给出可执行的补救方式：\n
   - 把仓库 clone 到本地并用 Cursor 打开；或\n
   - 提供文件路径与相关代码片段（含行号/关键上下文）。\n
3. 只有在看到源码/可复现实验后，才输出确定性结论；否则用“未验证/需要源码确认”。\n
